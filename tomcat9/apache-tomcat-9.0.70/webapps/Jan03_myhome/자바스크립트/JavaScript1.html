객체 접근 방법

		id속성 : 하나의 웹 페이지에 id는 하나만 가능하다
			id는 모든 태그에 부여할 수 있는 속성이다
			var ob = document.getElementsById("id속성");
			동일한 아이디가 여러개 있어도 첫번째 하나만 가져온다
		name 속성 : 서버로 정보를 전송 가능한 태그만 가질 수 있는 속성이다
			id와 달리 여러 태그가 같은 이름을 가질 수 있다
			배열 형태로 접근한다
			input, select, textarea만 가질 수 있다
		var obs = document.getElementsByName("name속성");
			<= 동일 name을 갖는 다른 태그가 없더라도 항상 배열 형태로 처리된다
			obs[0].value=4; 인덱스 생략을 할 수 없다.
			name 속성을 갖지 않는 태그들은 innerHTML 속성을 통해 내용을 변경할 수 있다
		class 속성 : 보통은 css를 지정하기 위해 사용한다. 배열로
			var obs = document.getElementsByClassName("class속성");
		태그명으로 접근 가능
			var obs = document.getElementsByTagName("tagname 속성");
		최근에 추가
			querySelector("#id" 또는 ".class")	-- 하나만 가져옴
			querySelectorAll("#id" 또는 ".class"태그나 name도 가능) -- 여러개 가져옴
			
		form태그를 이용해 객체 참조를 가져올 수도 있다.
		form태그 참조 가져오기
			1)document.form의 name속성;
			2)document.getElementById("form의 id속성");
			3)document.forms[0];	하나의 페이지에 form이 여러개 있을 때 document의 forms라는 배열 객체에 폼에 대한 참조가 저장되어 있다
			이 셋중 하나로 form객체를 가져올 수 있다	
			
			form객체.name속성 형태로 접근 가능하다
